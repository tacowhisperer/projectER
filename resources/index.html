Welcome to Project-ER!

Project-ER (Project Eat Rice) is a work-in-progress server that takes the weekly servery menus found here, and then converts them from PDF format into an easy-to-process JSON format.

The server is written in JavaScript using Node.js + Express + download-file + pdf2json. The source code for the service can be found here.

About Project-ER

Project-ER, namely Project Eat Rice, bega during HackRice 7.5 in 2018. Though the idea of providing a way to get menu information at Rice is not original, the devs noted that nobody had provided an easy way to access the weekly menu information from the Rice Dining website. More APIs are planned to be implemented in the near future, so stay tuned for those.
