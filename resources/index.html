Welcome to Project Eat Rice!

This is currently a work-in-progress server that takes the weekly servery menus found here, and converts them into easy-to-process JSON files, which are outlined in greater detail below.


API Usage

The list below shows the different GET requests and outputs of those requests. Note that keys in the sample JSON objects shown mean that they can be any valid JSON string (bold keys should be interpreted literally).

  • /WeeklyMenu: Returns an JSON object (application/json) of the weekly menu for all serveries that have a weekly menu PDF here. The format of this JSON object is as follows: 
    {
	"Servery Name 1": {
		"chef": "Chef Name",
		"menu": {
			"lunch": {
				"monday": [
					"Entreé 1",
					"Entreé 2",
					...
				],
				"tuesday": [
					"Entreé 1",
					"Entreé 2",
					...
				],
				"wednesday": [
					"Entreé 1",
					"Entreé 2",
					...
				],
				"thursday": [
					"Entreé 1",
					"Entreé 2",
					...
				],
				"friday": [
					"Entreé 1",
					"Entreé 2",
					...
				],
				"saturday": [
					"Entreé 1",
					"Entreé 2",
					...
				],
				"sunday": [
					"Entreé 1",
					"Entreé 2",
					...
				],
			},
			"dinner": Same format as "lunch"
		}
	},
	"Servery Name 2": Same format as "Servery Name 1",
	...
}
  • Other APIs are currently in the works so stay tuned.

If you can see this page, it means that the server is up and running as expected. However, if something is not behaving as it should be, send a message to as100@rice.edu or jcd7@rice.edu to fix it as soon as possible.
